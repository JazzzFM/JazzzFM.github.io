Evaluation grid


The main evaluation of this dataviz class is based on a real-life dataviz project. You have to set a ~3 people team, find a dataset, analyse it with a few insightful charts, build a report with R Markdown and host it on Github. You will be evaluated as follow:


General

  • Project originality
  • Project submitted on time
  • No major mistakes in the analysis
  • Redaction and clarity
  • General feeling

Dataviz

  • Appropriate charts
  • Explain and Justify chart choice
  • Chart general appearance
  • At least 2 different charts
  • Avoid main dataviz caveats

Code

  • Code is clean, easy to read
  • Code is commented
  • Link to source code and source data available

R Markdown

  • General appearance
  • Code is available
  • Link to source code and source data available
  • Reproducible

Github

  • Account is setup properly
  • Report is accessible online
  • Repository is documented, with url and read me
